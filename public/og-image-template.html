How to Save as OG Image:

  1. Right-click on the black box
  2. Select "Inspect" or press F12
  3. Right-click the .og-container element
  4. Select "Capture node screenshot"
  5. Save as og-image.png
  6. Move to /public folder

Or use a screenshot tool to capture exactly the black area (1200x630px)
